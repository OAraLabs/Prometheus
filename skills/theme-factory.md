# Theme Factory

A curated collection of professional font and color themes, each with carefully selected color palettes and font pairings. Themes can be applied to any file output generated by Prometheus.

## Purpose

Apply consistent, professional styling to generated outputs. Each theme includes:
- A cohesive color palette with hex codes
- Complementary font pairings for headers and body text
- A distinct visual identity suitable for different contexts

## Usage Instructions

1. **Present available themes**: Show the user the list of themes below with their descriptions
2. **Ask for their choice**: Which theme to apply
3. **Wait for selection**: Get explicit confirmation
4. **Apply the theme**: Use the selected theme's colors and fonts in the output via `file_write` or `file_edit`

## Application Process

After a theme is selected:
1. Read or generate the target output
2. Apply the specified colors and fonts consistently throughout
3. Ensure proper contrast and readability
4. Maintain the theme's visual identity across all sections

For HTML outputs, include Google Fonts links and apply CSS variables:
```html
<style>
  :root {
    --primary: #1A365D;
    --secondary: #2A6F97;
    --accent: #48CAE4;
    --font-heading: 'Merriweather', serif;
    --font-body: 'Source Sans Pro', sans-serif;
  }
</style>
```

## Create a Custom Theme

When none of the existing themes fit:
1. Gather inputs: mood, brand colors, industry, audience
2. Select complementary colors (primary, secondary, accent)
3. Choose font pairings (heading + body)
4. Present the theme for review
5. Apply after confirmation

Write the theme definition to the output file using `file_write` with the same CSS variable pattern.
